Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa perbedaan antara interface dengan class biasa?
2. Mengapa interface tidak memiliki implementasi method di dalamnya?
3. Apa tujuan penggunaan keyword "implements" pada sebuah class?
4. Apakah sebuah class dapat mengimplementasikan lebih dari satu interface? Jelaskan dan berikan contoh sederhana.
5. Jika sebuah class mengimplementasikan interface, apa yang terjadi jika class tersebut tidak mengimplementasikan semua method dari interface tersebut?

jawab
1. Perbedaan interface dengan class biasa:
   a. Interface hanya berisi deklarasi method (tanpa body), sedangkan class berisi implementasi lengkap
   b. Interface tidak bisa diinstansiasi, class bisa
   c. Class bisa memiliki constructor, interface tidak
   d. Interface mendukung multiple inheritance, class hanya single inheritance
   e. Interface menggunakan keyword implements, class menggunakan extends

2. interface tidak memiliki implementasi method karena tujuannya adalah untuk mendefinisikan kontrak atau blueprint yang harus diikuti oleh class yang mengimplementasikannya.
    Ini memungkinkan fleksibilitas dalam bagaimana method tersebut diimplementasikan.

3. Tujuan penggunaan keyword "implements" pada sebuah class adalah untuk menunjukkan bahwa class tersebut akan mengikuti blueprint yang telah ditetapkan oleh interface.
   Dengan menggunakan "implements", class tersebut wajib mengimplementasikan semua method yang dideklarasikan dalam interface.

4. Ya, bisa. Ini adalah cara Java mendukung multiple inheritance.
    interface A {
        void methodA();
    }
    interface B {
        void methodB();
    }
    class C implements A, B {
        public void methodA() {
            // implementasi methodA
        }
        public void methodB() {
            // implementasi methodB
        }
    }

5. Jika sebuah class mengimplementasikan interface tetapi tidak mengimplementasikan semua method dari interface tersebut, maka class tersebut harus dideklarasikan sebagai abstract.
   Jika tidak, akan terjadi error pada saat kompilasi karena class tersebut tidak memenuhi kontrak yang ditetapkan oleh interface.
